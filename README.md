## 🛠️ Troubleshooting

### The app will not open
- check that you finished the download
- unzip the file if needed
- try running it again
- move it to a normal folder like Desktop

### Windows shows a security prompt
- select the option that lets you open the app
- make sure you downloaded it from the release page
- try again after the file is fully downloaded

### The file does not load
- check that the file ends in `.yml` or `.yaml`
- make sure the file is not broken
- open a smaller YAML file to test

### The text looks wrong
- the file may use a format the app does not handle well
- try a clean YAML file
- check for tabs, odd spacing, or broken indentation

### The app is slow
- close other large apps
- use a smaller YAML file
- restart the app and try again
